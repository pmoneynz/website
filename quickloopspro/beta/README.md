# QuickLoopsPro Beta Downloads

This directory contains beta releases for QuickLoopsPro.

## Files

- `appcast.xml` - Sparkle appcast feed for automatic updates
- `QuickLoopsPro.dmg` - **Place your DMG file here**
- `release-notes/1.0.1.html` - Release notes for version 1.0.1

## Upload Instructions

1. Upload your `QuickLoopsPro.dmg` file to this directory
2. Update `appcast.xml` with the correct file size (in bytes) for the `length` attribute
3. Update `release-notes/1.0.1.html` with actual release notes
4. Ensure the DMG file is properly signed and notarized for macOS distribution

- Appcast: https://pmoneymusic.com/quickloopspro/beta/appcast.xml
- DMG: https://pmoneymusic.com/quickloopspro/beta/QuickLoopsPro.dmg
