Highlights

Dual-Deck Looper

Two independent loopers side-by-side — run Deck 1 and Deck 2 as separate performance lanes.

8 Loops Per Deck

8 loop slots per deck (16 total loops) so you can capture and recall ideas instantly.

Single/Multi Mode

Single: one loop at a time (new loop stops previous). Multi: multiple loops at once with tempo sync.

Simple Record/Play

Tap a loop to record. Tap again to play. Built for zero-friction capture in the moment.

Match Speed

Match one deck’s BPM to the other deck to keep both sides locked together.

MIDI Learn Mode

Learn MIDI mappings by pressing buttons — fast setup that feels like hardware.

Core Looping Features

The core workflow: record, play, sync, and see what’s happening in real time.

  • Dual-Deck Looper — Two independent loopers side-by-side
  • 8 Loops Per Deck — Each deck has 8 loop slots (16 total loops)
  • Record/Play — Tap loop to record, tap again to play
  • Single Mode — Only one loop plays at a time (new loop stops previous)
  • Multi Mode — Multiple loops can play simultaneously with tempo sync
  • Master Loop Sync — In Multi Mode, loops sync to master loop length (loop 1) with power-of-2 ratios (1/64× to 64×)
  • Loop Progress Visualization — Real-time progress bar showing playback position
  • Loop Information Display — Shows BPM, bar count, and duration for each loop
  • BPM Detection — Automatic tempo detection (80–160 BPM range)
  • Bar Count Display — Supports whole bars and fractional notes (½, ¼, ⅛, 1/16, 1/32, 1/64)
Loop Manipulation

Change loop length and speed live, with controls designed for performance and MIDI.

  • Halve Loop — Cut loop length in half (½ button)
  • Double Loop — Double loop length (x2 button), up to original recorded length
  • Speed Control — Playback speed adjustment from 0.5× to 2.0× via slider
  • Reset Speed — Return to 1.0× speed
  • Match Speed — Match one deck’s BPM to the other deck
  • Speed Up/Down — MIDI-mappable
  • Variable Speed Playback — Real-time speed changes with interpolation
Transport Controls

Fast stop/clear operations and visible state so you don’t get lost mid-set.

  • Stop — Stop current loop playback
  • Clear Mode — Enter clear mode, tap pad to delete
  • Clear All Loops — Triple-click Clear button to delete all loops instantly
  • Loop State Indicators — Visual feedback (empty, recording, playing, stopped)
  • Color-Coded Loops — 8 distinct colors for visual identification
Audio Features

Monitoring, gain staging, meters, and routing that fit real DJ/audio setups.

  • Input Monitoring — Hear input signal through output
  • Auto Mute Monitor — Automatically disable monitoring when playback starts
  • Input Level Control — Adjust recording input gain
  • Output Level Control — Adjust playback volume (up to 2.0×)
  • Master Volume — Global master volume control
  • Level Meters — Real-time input/output level meters
  • Multi-Channel Support — Per-deck input/output channel routing
  • Channel Routing — Select specific input/output channels per deck
Audio Device Management

Control devices and latency characteristics without leaving the app.

  • Input Device Selection — Choose audio input device
  • Output Device Selection — Choose audio output device
  • Buffer Size Control — Adjustable buffer size (32, 64, 128, 256, 512, 1024 frames)
  • Device Refresh — Refresh audio device list
  • Default Device Fallback — Automatic fallback to system defaults
MIDI Control

Hardware-style control: learn mappings fast, handle hot-plug, and keep decks independent.

  • MIDI Learn Mode — Learn MIDI mappings by pressing buttons
  • Performance Pad MIDI Mapping — Map each of 8 loop pads to MIDI notes
  • Transport MIDI Mapping — Map Stop, Clear to MIDI
  • Speed MIDI Mapping — Map Speed Up, Speed Down, Reset Speed, Match Speed
  • Loop Manipulation MIDI — Map Halve Loop, Double Loop
  • Per-Deck MIDI — Independent MIDI mappings per deck
  • MIDI Channel Support — Supports all 16 MIDI channels
  • MIDI Device Auto-Connect — Automatically connects to available MIDI sources
  • MIDI Device Hot-Plug — Handles MIDI device connection/disconnection
Visual Feedback

Instant state and meter feedback that’s readable during performance.

  • Deck Status Indicators — Toolbar indicators showing deck state (idle, recording, playing, overdubbing)
  • Loop Visual States — Icons and borders indicate loop state
  • Master Loop Badge — “M” badge on master loop in Multi Mode
  • Flash Feedback — Visual flash on MIDI/UI triggers
  • Color-Coded Loop Grid — 8 distinct colors matching MIDI settings
  • Horizontal Master Meter — Combined master output level meter in toolbar
  • Vertical Level Meters — Per-deck input/output meters
Settings & Configuration

Setup options live where they should, and they persist.

  • Persistent Settings — All settings saved and restored
  • Per-Deck Settings — Independent settings per deck
  • Settings Dialog — Comprehensive audio settings interface
  • MIDI Settings Dialog — Dedicated MIDI configuration interface
